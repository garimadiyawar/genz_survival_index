# Gen Z Survival Index

An experimental data project exploring which cities are most livable for Gen Z based on affordability, job opportunities, remote work availability, and quality of life indicators.

## Project Overview

This project combines multiple datasets including:

- Cost of living indices
- GDP per capita
- Youth unemployment
- Remote work statistics
- AI job postings
- Quality of life indicators

The goal is to build a **"Gen Z Survival Index"** that ranks cities based on how realistically a young person could live and work there.

## Tech Stack

- Python
- Pandas
- SQL
- Jupyter
- React (dashboard visualization)

## Project Structure

- `notebooks/` – data collection, cleaning, and analysis
- `data/` – processed datasets
- `dashboard/` – React visualization prototype
